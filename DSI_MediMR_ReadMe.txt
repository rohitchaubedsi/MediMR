There are various VR and AR components that work together for MediMR solution.

Software to download 
Unity 3D along with AR Library - Vuforia 

Hardware Required 
- HTC VIVE 
- Leap Motion sensors 
- Digital Camera ( as a subsitute of Microsoft HoloLens) 

Download the zip files (for MediMR.exe.7z file please download 7zip software - http://www.7-zip.org/) 
Unzip the files and run the executable file with .exe extension to play the VR and AR
